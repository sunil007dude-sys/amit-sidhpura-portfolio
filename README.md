# Amit N. Sidhpura — Portfolio Website

Personal portfolio landing page for Amit N. Sidhpura, Senior Manager — Sales, Operations & Training Development at the International Gemological Institute.

## Live Site
[amitsidhpura.com](https://amitsidhpura.com)

## Tech Stack
- Pure HTML & CSS (no frameworks, no build tools)
- Google Fonts (Cormorant Garamond, DM Sans, Cinzel)
- Vanilla JavaScript (scroll effects only)

## Deploy
Drag `index.html` into [Netlify](https://netlify.com) — done.
